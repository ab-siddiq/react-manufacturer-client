# What is used in this project?
- Node.js
    - For server side
- Express.js
    - for server side
- MongoDB
    - for database
- React.js
    - for frontend
- Tailwind Css
    - For Designing fronted
- Daisy Ui
    - For table, cart etc.
- Firebase
    - For authentication
    - For Hosting client side
- Heroku
    - For hosting server side
